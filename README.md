## Task 2: Student Grade Calculator

### Description

A Java console-based application that calculates a student's total marks, average percentage, and grade based on marks entered for multiple subjects.

The program accepts student marks, performs calculations, and displays the final result with the assigned grade.

### Features

- Accepts marks for multiple subjects
- Calculates total marks
- Calculates average percentage
- Determines grade based on performance
- Displays final result

### Concepts Used

- Variables
- Data types
- Scanner class
- User input handling
- Arithmetic operations
- Conditional statements
- Loops

# Task 3: ATM Interface

### Description

A Java-based ATM simulation application that demonstrates Object-Oriented Programming concepts.

The system contains a separate `BankAccount` class for managing account details and an `ATM` class for handling user interactions.

The ATM allows users to perform basic banking operations through a menu-driven interface.

### Features

- Check account balance
- Deposit money
- Withdraw money
- Validate transaction amounts
- Handle insufficient balance
- Interactive ATM menu
- Multiple transactions until exit

### Classes Used

### BankAccount.java

Responsible for:

- Storing account balance
- Deposit operation
- Withdrawal operation
- Balance checking

### ATM.java

Responsible for:

- Displaying ATM menu
- Taking user choices
- Calling account operations

### Main.java

Responsible for:

- Creating objects
- Starting ATM application

### Concepts Used

- Classes and Objects
- Encapsulation
- Constructors
- Object relationships
- Method calling
- Menu-driven programming

# Task 5: Student Management System

### Description

A Java console application designed to manage student records.

The system stores multiple student objects using `ArrayList` and provides complete CRUD functionality.

The application allows users to add, view, search, update, and delete student information.

---

## Features

### Add Student

Allows users to add new student records including:

- Student ID
- Admission Number
- Name
- Age
- Gender
- Grade
- Course
- Student Phone Number
- Parent Phone Number
- Email Address
- Permanent Address

---

### View All Students

Displays all student records stored in the system.

---

### Search Student

Searches for a student using the unique Student ID.

---

### Update Student

Allows modification of existing student details.

---

### Delete Student

Removes student records from the system using Student ID.

---

## Classes Used

### Student.java

Represents student information and contains:

- Student attributes
- Constructor
- Getter methods
- Setter methods
- Display method

### StudentManagementSystem.java

Handles:

- Adding students
- Viewing students
- Searching students
- Updating students
- Deleting students

### Main.java

Provides:

- User menu
- User input handling
- Application control

# How to Run the Projects

## Requirements

Before running the projects, make sure you have:

- Java Development Kit (JDK 8 or above)
- Java IDE or code editor
  - VS Code
  - IntelliJ IDEA
  - Eclipse

---

## Steps

 1. Clone the repository

    git clone https://github.com/keerthipriya7733-droid/CODSOFT.git

2. Open the required project folder

Example:

3. Compile the Java file
javac Main.java
4. Run the program
java Main
